# Kiss a Friend! (Forge)

A 1-to-1 Forge port of the Minecraft 1.20.1 version of
[Kiss a Friend!](https://modrinth.com/mod/kiss-a-friend), with the Create: Estrogen
interoperability removed. Everything else behaves the same as the original: sneak while
you and another player look at each other to share a kiss and some heart particles.

## Why this exists

I made this for a private server running a private modpack, for a group of friends. The
original mod is Fabric-only and we're on Forge, so I ported it. You're welcome to use it
too. That's it.
